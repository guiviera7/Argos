# Projeto Argos

Ferramenta de gestão de estoque que fornece visibilidade 
e previsibilidade sobre a área de picking e estoque.

## O problema

Antes do Argos, não havia nenhuma forma prática e direta 
de visualizar o que estava crítico na área de separação 
e no estoque. A tomada de decisão dependia de conferência 
física manual e de dados brutos do SAP sem tratamento.

## O que o Argos faz

Integra dados exportados do SAP e apresenta em duas abas:

- **Estoque** — visibilidade do estoque pulmão
- **Área de Separação** — foco na área de picking com 
  saldo, quantidade de SKUs, lotes, código de SKU e 
  informações para tomada de decisão estratégica

## Como usar

1. Exporte os dados do SAP em formato CSV (UTF-8)
2. Cole as duas planilhas nos campos indicados
3. Navegue pelas abas para visualizar o status atual

## Tecnologias

- Microsoft Excel
- SAP (fonte de dados)
